Project 1 ✅
🚀 **New Machine Learning Project: Mine or Stone Detection**


Sonar Detection System is a machine learning classification project that predicts whether an object detected by sonar signals is a Rock or a Mine. The model is trained on the Sonar dataset containing 60 numerical features representing the energy of sonar signals reflected at different frequencies. Data preprocessing, exploratory data analysis (EDA), feature scaling, and model training were performed using Python and scikit-learn. Multiple classification algorithms were evaluated, and hyperparameter tuning was applied to improve performance. The final model can classify sonar signal inputs and assist in distinguishing underwater mines from rocks based on reflected acoustic signals.


Project 2 ✅
🚀 **New Machine Learning Project: Breast Cancer  Detection**


Breast Cancer Detection System is a machine learning classification project that predicts whether a tumor is Benign or Malignant based on diagnostic medical features. The model is trained on a breast cancer dataset containing numerical attributes extracted from cell nuclei measurements. Data preprocessing, exploratory data analysis (EDA), feature scaling, and model training were performed using Python and scikit-learn. Multiple classification algorithms were evaluated, and hyperparameter tuning was applied to improve performance. The final model can analyze patient diagnostic data and assist in the early detection of breast cancer by accurately classifying tumors.


Project 3 ✅
🚀 **New Machine Learning Project: Credit Card Fraud Detection**



In this project, I built a classification model to identify whether a credit card transaction is legitimate or fraudulent using transaction data. Since fraud detection datasets are highly imbalanced, I explored preprocessing techniques and evaluated multiple machine learning models to improve detection performance.

### 🔹 What I worked on:

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Handling Imbalanced Data
* Feature Scaling
* Model Training and Evaluation
* Hyperparameter Tuning using Cross-Validation
* Performance Evaluation using Precision, Recall, F1-Score, ROC-AUC, and PR-AUC

### 🛠️ Technologies Used:

* Python
* Pandas
* NumPy
* Matplotlib & Seaborn
* Scikit-learn

This project helped me gain a deeper understanding of classification problems, model evaluation on imbalanced datasets, and the importance of selecting appropriate metrics for fraud detection.
